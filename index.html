Rules for Blackjack!

Each card has different values. Cards 2-10 has same value as their number. Cards Jack, Queen, and King has value of 10. Ace has value of 11. Goal is to hit 21 with max of 5 cards. If total value exceeds 21, the hand is bust and result in automatic loss.

Hit button will draw new card for player.

Stand button will pass the turn to dealer

Duel button will start new round.
